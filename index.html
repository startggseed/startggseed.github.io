Instructions

Paste the event link with format "https://www.start.gg/tournament/_____/event/_____"
Look below for instructions on api keys
For the Spreadsheet, upload a .csv with any two columns labeled "Phase Seed" and "Seed ID"
If there are no column labels, the first column will be treated as Phase Seed, and the second column as Seed ID

To get a start.gg api key:
1. Go to start.gg
2. Click your profile icon in the bottom left and choose "Developer Settings"
3. Press "Create new token"
4. Give it a description and click save
5. Copy your token to the "start.gg api key" entry above
